# Features Research: Beautiful Simple TODO CLI

## Table Stakes (Must Have)
- **Add**: `bst add "Buy groceries"`
- **List**: `bst list` (with fancy formatting)
- **Done**: `bst done 1` (by index)
- **Remove**: `bst remove 1`

## Differentiators (Visual Polish)
- **Status Symbols**: Checkmarks for done, bullet points for pending.
- **Color Coding**: Green for completed, blue/cyan for active.
- **Header/Footer**: A nice border when listing tasks.

## Anti-Features
- No database required (Keep it to JSON).
- No complex tagging/prioritization (Keep it simple).
